# spring-security

spring security 项目示例库

&emsp;&emsp;Spring Security 介绍，它的前身是Acegi Security，在被收纳为Spring 子项之后正式的更名为Spring Security。

&emsp;&emsp;应用程序的安全性主要体现在两个方面：认证和授权

&emsp;&emsp;认证是确认某主体在某系统中是否合法、可用的过程。这里的主体既可以是登陆系统的用户，也可以是接入的设备或者是其他系统。

&emsp;&emsp;授权是指当主体通过认证之后，是否允许其执行某项操作的过程。

&emsp;&emsp;这些概念并不是Spring Security独有的，而是整个的应用安全的基本关注点。
